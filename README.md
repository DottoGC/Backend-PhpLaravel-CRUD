# Backend basico (API)

## Instalacion
Para instalar todas las dependencias ejecutar
> composer install


## Base de datos
Tener un schema listo y la instancia de mysql con las siguientes caracteristicas
Schema name: energyresource
Host: localhost:3306
user and password: root

### Inicializar tablas 
> php artisan migrate


### iniciar servicio 
> php artisan serve
